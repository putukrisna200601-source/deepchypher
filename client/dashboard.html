Enkripsi & Sisipkan Pesan

Sembunyikan pesan teks ke dalam gambar PNG menggunakan AES-GCM dan LSB steganography. Buat secret key sekali pakai, lalu unduh stego-image untuk dibagikan secara terpisah.

Mulai Enkripsi
Gunakan gambar PNG asli. Hindari kompresi atau unggah ulang setelah payload disisipkan.

Buka & Dekripsi Pesan

Unggah stego-image PNG, masukkan secret key pesan, lalu buka pesan tersembunyi. Jika dekripsi berhasil, payload akan dianggap consumed dan tidak dapat dibuka ulang.

One-Time Reveal Secret Key Required Payload Validation Consumed After Success
  • unggah stego-image PNG
  • ekstrak payload
  • masukkan secret key pesan
  • tampilkan pesan jika valid
Mulai Dekripsi
Pastikan file PNG tidak diubah, dikompresi, atau diunggah ulang agar payload tetap valid.

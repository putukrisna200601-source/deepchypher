Setel Ulang Akses Akun

Buat password akun baru dan master key akun baru untuk mengamankan kembali dashboard DeepChypher Anda.

Yang sedang Anda setel ulang

  • Password akun → untuk autentikasi saat sign in
  • Master key akun → untuk membuka dashboard setelah login berhasil
Secret key pesan tidak diatur ulang dari halaman ini.
Password terlalu lemah. Penuhi minimal 3 syarat keamanan. Digunakan untuk autentikasi saat sign in.
Konfirmasi password tidak cocok.

Saran password akun

  • minimal 8 karakter
  • gunakan kombinasi huruf besar, huruf kecil, angka, dan simbol
  • jangan gunakan password lama
  • jangan samakan dengan master key akun
Master key terlalu lemah. Penuhi minimal 3 syarat keamanan. Master key digunakan untuk membuka dashboard akun setelah login berhasil.
Konfirmasi master key tidak cocok. Master key tidak boleh sama dengan password akun.

Tentang master key akun

  • digunakan untuk membuka dashboard akun
  • berbeda dari password akun
  • jangan dibagikan ke orang lain
  • sebaiknya tidak sama dengan password akun lama maupun password akun baru
